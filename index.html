$ vimtutor

Normal Mode

Moving around the document

Insert Mode

Only when typing

Visual Mode

Selecting text

Movement

h, j, k, l left, down, up, right
w word
b back
e end (of word)
$ end of line
^ beginning of line
f{char} forward to {char}
t{char} forward up to {char}

Movement

gg top of file
G bottom of file
{number}G go to that line
Shift + H/M/L move around the screen (high/middle/low)

Verbs

Character Action
c change
d delete
v visually select
y yank (copy)
p put (paste)
. repeat
u undo

More verbs

Character Action
a append
s substitute
x delete character
r replace character

Shortcuts

Character Action
A ($a) append at the end of the line
I (^i) insert at the beginning of the line
C (c$) change till the end of the line
D (d$) delete to the end of the line

Putting It Together

Keystrokes Translation
d$ delete to the end of the line
c2w change 2 words
da" delete around "
ci" change inside quotes
5p put yanked text into buffer 5 times
3dd delete current line three times

Searching

/ search for text
n, N go to next/previous result
* search for this word
:%s/[find]/[replace]/gc subsitute [find] with [replace]

:commands

Command Action
:w [file] write to [file]
:e [file] edit [file]
:sp [file] open [file] in split window
:vs [file] open [file] in vertically split window
:tabedit [file] open [file] in a new tab
:!mkdir foo run stuff in your shell
:make run your makefile

Macros

qa[keystrokes]q save keystrokes to "a"
3@a replay the keystrokes in "a" 3 times
@@ replay last macro

Misc

zz center screen on cursor
o, O new line below (or above) cursor, insert mode
J join this line with the one below it
